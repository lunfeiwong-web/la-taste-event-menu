La Taste x 3 Yue Static Event Menu Website

Upload everything in this folder to Netlify or GitHub Pages.

Files:
- index.html: the website page
- styles.css: visual styling
- app.js: renders the page and opens WhatsApp enquiry messages
- data/menu-data.json: edit packages, menu sections, terms, contact details and gallery list here
- assets/images: compressed WebP images

No database, backend, login, payment gateway or framework is required.

For WhatsApp, edit these fields in data/menu-data.json:
- contact.mainWhatsApp
- contact.secondWhatsApp

Use international format without plus sign, for example 60125533400.
